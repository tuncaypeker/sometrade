SomeTrade
Advanced .NET algorithmic trading framework

Overview
SomeTrade is a modular algorithmic trading framework built with .NET.
It includes a flexible indicator engine, harmonic pattern tools, price-action swing mapping, a visual strategy builder, and a complete backtesting pipeline.
The project is designed for developers who want to create, test, and automate trading strategies with clean architecture and extendable modules.

Features
✔ Modular Indicator Engine – Build and combine custom indicators easily
✔ Harmonic Pattern Detection – Identify common harmonic structures
✔ Price-Action Swing Mapping – Automatic swing, zone and structure analysis
✔ Strategy Builder – Compose rule-based strategies visually or with code
✔ Full Backtesting Engine – Historical tests, reporting and analytics
✔ Live Execution Support – Connect your strategies to live markets
✔ Clean .NET Architecture – Extensible, testable, and production-ready
